SMART CITY BUSINESS CASES (21–30)

21. Smart Water Distribution & Leak Detection

Problem Statement

The city’s water network must maintain equitable distribution while detecting leaks, abnormal usage and pipeline bursts in real-time.

Data Structures & C++ Code


#include <vector>
#include <unordered_map>

struct PipeSegment {
    int id;
    double flow_rate;     // L/s
    double pressure;      // psi
    bool leak_suspected;
};

using WaterNetwork = std::unordered_map<int, PipeSegment>;

Application Logic

Compare predicted vs actual flow using pressure profiles. If a sudden drop occurs, flag the segment as leak-suspected and auto-adjust valves.

Complexity

Pipe update O(1); network-wide balancing O(N).

22. Waste Collection Optimization

Problem Statement

Optimize garbage truck routing, predict bin fill levels and reduce fuel/time cost.

Data Structures & C++ Code


#include <queue>
#include <vector>

struct Bin {
    int id;
    double fill_level; // 0–1
    bool recyclable;
};

using PickupQueue = std::priority_queue<std::pair<double,int>>;

Application Logic

Sensors push bins into PQ. Route planner selects bins above threshold and generates an A*/Dijkstra route.

Complexity

PQ insertion O(log N); routing O(E log V).

24. Emergency Response & Dispatch

Problem Statement

Dispatch nearest emergency units using live traffic and incident severity.

Data Structures & C++ Code


#include <vector>
#include <string>

struct Unit {
    int id;
    double lat, lon;
    bool available;
    std::string type; // ambulance, fire, police
};

using UnitList = std::vector<Unit>;

Application Logic

Compute nearest available unit via haversine distance and traffic-based routing.

Complexity

Nearest search O(N); routing O(E log V).
