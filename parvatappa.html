21. Solar Farm Load Balancing & Energy Forecasting

Problem Statement

City solar farms must forecast generation and distribute load to avoid grid instability and maximize storage usage.

Data Structures & C++ Code

#include <vector>
#include <map>

struct SolarPanel {
    int id;
    double current_output; // kW
    double temperature;
    double irradiation;
};

using SolarFarm = std::map<int, SolarPanel>;

Application Logic

Use irradiation + temperature to forecast power. Distribute output to storage units and demand nodes using dynamic balancing.

Analysis

Forecasting per panel O(1). Farm-level balancing O(N log N) for N panels.

22. Smart Parking with Dynamic Pricing

Problem Statement

Parking spots should adjust pricing based on demand, occupancy and location.

#include <unordered_map>

struct ParkingSpot {
    int id;
    bool occupied;
    double base_price;
    double surge_factor; // dynamic multiplier
};

using ParkingMap = std::unordered_map<int, ParkingSpot>;

Application Logic

Compute surge_factor = f(occupancy ratio). Update pricing every 5 minutes. Show availability to drivers.

Analysis

Map updates O(1). Surge computation O(N).

23. Emergency Dispatch Optimization

Problem Statement

Ambulance, fire and police require fastest dispatch using real-time traffic routing.

#include <queue>
#include <vector>

struct DispatchUnit {
    int id;
    double lat, lon;
    bool available;
};

using UnitList = std::vector<DispatchUnit>;

Application Logic

Pick nearest available unit + shortest path via Dijkstra on road network. Include hospital load balancing.

Analysis

Routing O(E log V). Selection O(N).
